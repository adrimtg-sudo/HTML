Formato de texto

El formato de texto se dota con código CSS preferiblemente, siempre cuando se pueda.
"b"" es negrita o Bold, pero es preferible usar CSS ya que es más correcto.
"Strong" actúa como negrita pero es más correcto que usar Bold.
Con i podemos escribir cursiva o itálica, aunque esto también se debería hacer con CSS.
Con "em" podemos escribir texto enfatizado que resalta la importancia de una palabra o sentencia.
Parecido a lo anterior, podemos usar "mark"para resaltar texto importante como un subrayador.
Si alguna vez debemos corregir, podemos usar "del" para tachar palabras y frases.
Para representar texto que ha sido añadido usamos "ins" de manera que el texto aparezca así y es correcto usarlo ya que aporta significado.
Por último tenemos los subíndices y superíndices, su uso probablemente más notado en matemáticas o anotaciones científicas: para poner subíndices usamos "sub" como por ejemplo H2O, CO2, etc.
Para poner superíndices usamos "sup" como por ejemplo MC2 o 22=4.
