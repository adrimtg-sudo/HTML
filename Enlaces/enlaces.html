Enlaces o Links.

Usamos "a" para implementar un enlace dentro de un texto como es el caso.
En "href" debemos poner la web a la que queremos que nos direccione al pulsar el enlace. Hemos usado también "target" para definir donde se abre la página al clickar el enlace; Blank es en otra ventana o pestaña.
Las otras opciones son "Self" para la misma ventana o pestaña, "Parent" para el marco primario y "Top" para el cuerpo completo de la ventana.
